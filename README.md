# Bloggin

[Live Demo](https://bloggin-app-six.vercel.app)

## Overview

Bloggin is a full stack web application for creating and sharing blog content. It focuses on performance, clean UI, and real time data handling.

## Key Features

- User authentication with secure login and signup  
- Create, edit, and manage blog posts  
- Real time data updates using cloud database  
- Responsive interface for mobile and desktop  
- Light and dark mode support  

## Tech Stack

**Frontend**
- Next.js with React  
- TypeScript  

**Styling**
- Custom CSS with theme variables  
- Responsive layout and dark mode support  

**Backend and Database**
- Firebase Authentication  
- Cloud Firestore  

## Frontend Focus Areas

- Component based architecture with reusable UI  
- Type safe development  
- State management for dynamic content  
- Responsive and accessible design  
- Backend integration  

## Project Highlights

- Built a scalable blog platform with real time data flow  
- Implemented authentication and protected routes  
- Designed a consistent and user friendly interface  
- Structured code for maintainability
